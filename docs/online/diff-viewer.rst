Diff Viewer
===========

Keyboard Shortcuts
------------------

**Column 1 descriptions**

  *-* The path has been removed

  *+* The path has been created

  *M* The path has been modified

  *R* The path has been renamed

**Column 2 descriptions**

  *B* Block device

  *C* Character device

  */* Directory

  *>* Door

  *|* Named pipe

  *@* Symbolic link

  *P* Event port

  *=* Socket

  *F* Regular file
